# 第七题 取近似值
## 描述
    写出一个程序，接受一个正浮点数值，输出该数值的近似整数值。
    如果小数点后数值大于等于5,向上取整；小于5，则向下取整。

    输入描述：
    输入一个正浮点数值

    输出描述：
    输出该数值的近似整数值

    示例1
    输入：
    5.5

    输出：
    6
    
## 分析
    1，对数据进行取整操作，使用(int)方法，只保留整数部分，
    结果为i 
    2，然后判断d-i和0.5的关系，如果大于0.5，就返回i+1，
    否则返回i

## 总结
    双精度浮点型是double,不是Double
    强制类型转换是(int)
