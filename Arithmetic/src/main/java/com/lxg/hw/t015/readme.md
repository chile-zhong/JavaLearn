# 第十五题 求int型正整数在内存中存储1的个数
## 描述
    输入一个int型的正整数，计算出该int型数据在内存中存储时1的个数。

    输入描述：
    输入一个整数（int类型）

    输出描述：
     这个数转换成2进制后，输出1的个数
    
    示例1
    输入：
    5

    输出：
    2

## 分析
    将数字转换成二进制
    计数二进制中1的个数
