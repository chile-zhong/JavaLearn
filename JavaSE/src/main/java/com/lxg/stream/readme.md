# IO流
## IO流概述和分类
>IO流介绍
> - IO：输入/输出(Input/Output)
> - 流：是一种抽象概念，是对数据传输的总称。也就是说数据在设备间的传输称为流，流的本质是数据传
输
> - IO流就是用来处理设备间数据传输问题的。常见的应用：文件复制；文件上传；文件下载

> IO流的分类
>  - 按照数据的流向
>  1. 输入流：读数据
>  2. 输出流：写数据
>  
>  - 按照数据类型来分
>  1. 字节流        
>  字节输入流     
>  字节输出流
>  2. 字符流        
>  字符输入流     
>  字符输出流

> IO流的使用场景      
1.如果操作的是纯文本文件，优先使用字符流       
2.如果操作的是图片、视频、音频等二进制文件。优先使用字节流      
3.如果不确定文件类型，优先使用字节流。字节流是万能的流
